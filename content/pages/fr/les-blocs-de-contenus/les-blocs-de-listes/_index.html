L’utilisateur peut ajouter une mise en avant d'une sélection de pages, avec plusieurs options d'affichage. L’utilisateur rentre les champs suivants pour ajouter un bloc pages :

  • Titre
  • Publié ou non
  • Choisir la mise en page : Grille / Liste / Cartes
  • Type de liste : choix entre une sélection spécifique de pages / La page principale et ses enfants
  • Options d’affichage : Affichage de la description courte de la page principale / Afficher les descriptions courtes des pages / Afficher les images des pages
  • Page principale : Sélection d’une page via une liste déroulante
  • Sélectionner une ou plusieurs pages : liste déroulante
  • Possibilité de retirer une ou plusieurs pages

L’utilisateur peut ajouter une liste d'actualités mises en avant. L’utilisateur rentre les champs suivants pour ajouter un bloc Actualités : 

  • Titre
  • Publié ou non
  • Quelles actualités afficher : Toutes les actualités / Une catégorie spécifique / Une sélection d’actualités
  • Quantité d’articles : champ libre

L’utilisateur peut ajouter un organigramme présentant des personnes et leur fonction. L’utilisateur rentre les champs suivants pour ajouter un bloc organigramme : 

  • Titre
  • Publié ou non
  • Description
  • Avec des liens cliquables vers les pages des personnes ?
  • Avec un portrait de chaque personne ?

Ajouter une ou plusieurs personne : choisir la personne et son rôle (listes déroulantes)

Possibilité de retirer une ou plusieurs personnes
