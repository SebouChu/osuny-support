L’utilisateur peut ajouter des chiffres clés avec une jolie animation. Chaque chiffre peut être doté d'une unité et d'un texte.

L’utilisateur rentre les champs suivants pour ajouter un bloc chiffre clé :

  • Titre
  • Publié ou non
  • Texte
  • Ajouter un ou plusieurs chiffre clé : numéro / unité / description
  • Possibilité de retirer un ou plusieurs chiffres clé

L’accessibilité du bloc est indiqué. Sur chaque bloc, l’utilisateur peut supprimer le bloc et accéder aux dépendencies.

L’utilisateur peut ajouter une galerie d'images, avec pour chaque image une description accessible, un texte et un crédit. L’utilisateur rentre les champs suivants pour ajouter un bloc galerie :

  • Titre
  • Publié ou non
  • Description 
  • Mise en page : choisir la mise en page Grille ou Caroussel
  • Ajouter une ou plusieurs image : choix du fichier / texte alternatif/ Crédit / Intégrer le texte
  • Possibilité de retirer un ou plusieurs images

L’accessibilité du bloc est indiqué. Sur chaque bloc, l’utilisateur peut supprimer le bloc et accéder aux dépendencies.
