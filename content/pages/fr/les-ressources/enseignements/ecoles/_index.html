Possibilité de réinitialiser le filtre.

Au clic sur le nom de l’école, accès à la fiche de l’école sélectionnée. Possibilité de gérer l'équipe de l’école, de modifier et/ou supprimer la fiche.

Au clic sur gérer l'équipe : accès aux membres de l’équipe avec les rôle de chacun. Possibilité de modifier, supprimer et ajouter un rôle.

Au clic sur un rôle, l’utilisateur arrive sur la fiche du rôle.

notes: >- image: id: "87381fb6-bcc4-4c27-89f4-5cf5a616dc7b" file: "87381fb6-bcc4-4c27-89f4-5cf5a616dc7b" alt: >- credit: >- - template: chapter title: >- Accès à la fiche rôle position: 6 data: text: >-

Au clic sur un rôle, l'utilisateur arrive sur la fiche du rôle.
Possibilité de retirer une ou plusieurs personnes de chaque rôle défini.

Un message de confirmation apparait avant le retrait définitif

Possibilité de modifier un rôle : 
L’utilisateur peut changer la description du rôle 
L’utilisateur peut sélectionner des personnes via une liste déroulante
L’utilisateur peut ajouter des personnes
L’utilisateur peut retirer des personnes

- nformations L’utilisateur peut changer la description du rôle - Personnes L’utilisateur peut sélectionner des personnes via une liste déroulante L’utilisateur peut ajouter des personnes L’utilisateur peut retirer des personnes

- Supprimer L’utilisateur a la possibilité de supprimer un rôle. Un message de confirmation apparait avant la suppression définitive. - Ajouter L’utilisateur a la possibilité de rajouter un rôle dans l’équipe Il remplit le champ description. Il peut ensuite ajouter des personnes grâce à une liste déroulante.
