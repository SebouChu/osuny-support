Possibilité de créer un site web en remplissant les champs suivants : 

  • Informations
    • Nom
    • Url
    • Type de site (liste déroulante)
    • École
    • Langue
  • Git
    • Github
    • Point d'accès Git
    • Access token
    • Repository
  • Analytics
    • Plausible url
