L'utilisateur a accès à l'arborescence du site avec les dossiers et les sous dossiers.

À côté d'un dossier plein, il est indiqué "ouvrir le dossier". Au clic les sous dossiers ou sous pages apparaissent.

notes: >- image: id: "4062c6c4-cc94-4781-96d8-94d47122b80e" file: "4062c6c4-cc94-4781-96d8-94d47122b80e" alt: >- credit: >- - template: chapter title: >- Clic sur une page position: 4 data: text: >-

Au clic sur une page, les éléments essentiels sont indiqués :

  • Contenu 
    Le chapô : possibilité d'insérer une phrase pouvant illustrer la page
  • Blocs de contenu
    Voir section "Blocs de contenus"
  • Accessibilité
    Une information sur l'accessibilité de la page est indiquée
  • Informations
    - Publié ou non
    - Slug
    - Path
    - Page parente
    - Pleine largeur ou non
