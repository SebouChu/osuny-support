L'utilisateur a accès à l'arborescence du site avec les dossiers et les sous dossiers.

À côté du nom du dossier est indiqué "ouvrir le dossier". Au clic les sous dossiers et/ou sous pages apparaissent.

Au clic sur une page, les éléments essentiels sont indiqués :

  • Contenu 
    Le chapô : possibilité d'insérer une phrase pouvant illustrer la page
  • Blocs de contenu
    Voir section "Blocs de contenus"
  • Accessibilité
    Une information sur l'accessibilité de la page est indiquée
  • Informations
    - Publié ou non
    - Slug
    - Path
    - Page parente
    - Pleine largeur ou non

Création d'une page dans l'arborescence en remplissant les champs suivants :

  • Contenu : Titre / Nom affiché dans le fil d’ariane / Chapô / Titre affiché dans le header
  • Informations : Publié ou non / Slug / Page parente : liste déroulante / Classe CSS spécifique / Pleine largeur ou non
  • SEO : Méta description
  • Image : Choisir un fichier / Texte alternatif / Crédit / Chercher une image (Unsplash)
