L'utilisateur a accès aux éléments qui constituent le menu sélectionné

notes: >- image: id: "97eb01fc-1c99-4634-9d63-69c09d1030fd" file: "97eb01fc-1c99-4634-9d63-69c09d1030fd" alt: >- credit: >- - template: chapter title: >- Créer un menu position: 3 data: text: >-

L'utilisateur peut créer un menu en rentrant un titre et choisir un type de menu : 

  • Titre intermédiaire
  • Url
  • Page spécifique

Une fois le menu créer, il va le retrouver dans l'arborescence de son menu sur la page d'accueil du menu dans "primary".

notes: >- image: id: "6ec05fc8-c826-45e1-84f9-c23a182e97ef" file: "6ec05fc8-c826-45e1-84f9-c23a182e97ef" alt: >- Créer un menu credit: >- - template: chapter title: >- Modifier le menu position: 4 data: text: >-

L'utilisateur modifie juste les informations essentielles ici : le titre et les identifiants.
