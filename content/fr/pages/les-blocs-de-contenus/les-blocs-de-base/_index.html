L’utilisateur peut ajouter une section de document avec un titre, un texte, des notes de bas de page et une image.

L’utilisateur rentre les champs suivants pour ajouter un bloc : 

  • Titre
  • Texte
  • Notes
  • Publié ou non
  • Image : choisir un fichier
  • Texte alternatif
  • Crédit image

L'utilisateur peut ajouter un bloc image : une image avec sa description alternative accessible, son texte et son crédit.

L'utilisateur rentre les champs suivants pour ajouter un bloc image : 

  • Titre
  • Image : choisir un fichier
  • Publié ou non
  • Texte alternatif
  • Crédit
  • Texte image

L’utilisateur peut ajouter un bloc vidéo : une vidéo intégrée dans la page avec la transcription et sans lecture automatique. L’utilisateur rentre les champs suivants pour ajouter un bloc vidéo :

  • Titre
  • Url de la vidéo
  • Titre de la vidéo
  • Publié ou non
  • Transcription textuelle du contenu

L’utilisateur peut ajouter un bloc tableau : un tableau de données, aussi responsive et accessible que possible.

L’utilisateur rentre les champs suivants pour ajouter un bloc tableau :

  • Titre
  • Publié ou non
  • Ajout de colonne et de ligne
  • Transcription textuelle
