L’utilisateur peut ajouter une liste de fichiers téléchargeables, présentés avec leur poids. L’utilisateur rentre les champs suivants pour ajouter un bloc fichiers :

  • Titre
  • Publié ou non
  • Description
  • Ajouter un ou plusieurs fichiers : titre / choisir un fichier
  • Possibilité de retirer un ou plusieurs fichiers

L’utilisateur peut ajouter une liste de termes et leur définition. Un genre de dictionnaire, finalement. L’utilisateur rentre les champs suivants pour ajouter un bloc définitions :

  • Titre
  • Publié ou non
  • Ajouter une ou plusieurs définitions : titre / texte
  • Possibilité de retirer une ou plusieurs définitions

L’utilisateur peut ajouter du code HTML brut, permettant d'intégrer des cartes ou divers composants en iframe. 

L’utilisateur rentre les champs suivants pour ajouter un bloc html :

  • Titre
  • Publié ou non
  • Code du bloc d’intégration HTML
  • Transcription textuelle du contenu

L’utilisateur peut ajouter des coordonnées et des horaires d'ouverture, tout ce qu'il faut pour être contacté. L’utilisateur rentre les champs suivants pour ajouter un bloc contact : 

  • Titre
  • Publié ou non
  • Adresse postale : Nom /Adresse /Code postal /Ville/ Pays
  • Téléphones : Ajouter un ou plusieurs numéros / Possibilité de retirer un plusieurs numéros
  • Mails : Ajouter un ou plusieurs mails / Possibilité de retirer un plusieurs mails
  • Horaires : Ajouter un ou plusieurs créneaux horaires / jour ,matin , après midi / Possibilité de retirer un ou plusieurs créneaux horaires
