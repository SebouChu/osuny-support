L’utilisateur peut ajouter une galerie d'images, avec pour chaque image une description accessible, un texte et un crédit. L’utilisateur rentre les champs suivants pour ajouter un bloc galerie :

  • Titre
  • Publié ou non
  • Description 
  • Mise en page : choisir la mise en page Grille ou Caroussel
  • Ajouter une ou plusieurs image : choix du fichier / texte alternatif/ Crédit / Intégrer le texte
  • Possibilité de retirer un ou plusieurs images

L’utilisateur peut ajouter un titre, un texte, un visuel et 3 boutons pour inviter les personnes à agir. L’utilisateur rentre les champs suivants pour ajouter un bloc appel à action :

  • Titre
  • Publié ou non
  • Texte
  • Image : choisir un fichier
  • Texte alternatif
  • Crédit
  • Ajouter un ou plusieurs boutons d’action : texte / lien / ouvrir dans une nouvelle fenêtre
  • Possibilité de retirer un ou plusieurs boutons d’action

L’utilisateur peut ajouter un ou plusieurs témoignages, avec le texte, l'auteur, sa fonction et sa photo. L’utilisateur rentre les champs suivants pour ajouter un bloc témoignages : 

  • Titre
  • Publié ou non
  • Texte
  • Ajouter un ou plusieurs témoignages : texte/ auteur.e du témoignage/ métier de l’auteur.e / Image (choisir un fichier)
  • Possibilité de retirer un ou plusieurs témoignages

L’utilisateur peut ajouter une liste d'événements avec leur description, présentés en frise. L’utilisateur rentre les champs suivants pour ajouter un bloc frise :

  • Titre
  • Publié ou non
  • Choix de la mise en page : vertical ou horizontal
  • Ajouter un ou plusieurs événements : titre / texte
  • Possibilité de retirer un ou plusieurs événements
