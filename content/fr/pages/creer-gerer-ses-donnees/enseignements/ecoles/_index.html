Possibilité de créer une école avec les champs suivants :

  • Informations (Nom, téléphone, adresse, code postal, ville, pays, site web, logo
  • Formations dispensées : sélection des formations dispensées en cochant
notes: >- image: id: "854c8941-95c6-44cc-a609-74794704c82b" file: "854c8941-95c6-44cc-a609-74794704c82b" alt: >- credit: >- - template: chapter title: >- Accès à la fiche position: 3 data: layout: no_background text: >-

Au clic sur le nom de l’école, accès à la fiche de l’école sélectionnée avec ses informations, les formations dispensées, les sites web associés et la liste des membres de l'équipe.

L'utilisateur.rice a la possibilité de modifier ou de supprimer la fiche mais également de gérer l'équipe.

notes: >- image: id: "e627d4cb-3965-4a55-8461-34ffe9a2f3b3" file: "e627d4cb-3965-4a55-8461-34ffe9a2f3b3" alt: >- credit: >- - template: chapter title: >- Gérer l'équipe position: 5 data: layout: no_background text: >-

Au clic sur gérer l'équipe, l'utilisateur.rice accède à la liste des membres de l’équipe avec les rôle de chacun. Possibilité de modifier ou de supprimer une fiche.

Au clic sur un rôle, l’utilisateur.rice arrive sur la liste des membres ayant ce rôle là dans l'équipe. Il peut retirer des personnes si elle ou il le souhaite en cliquant sur "Retirer". Sur cet espace en cliquant sur "Modifier", l'utilisateur.rice peut modifier le titre du rôle et ajouter des personnes ayant ce rôle là en cliquant sur "Ajouter".

notes: >- image: id: "87381fb6-bcc4-4c27-89f4-5cf5a616dc7b" file: "87381fb6-bcc4-4c27-89f4-5cf5a616dc7b" alt: >- credit: >- - template: chapter title: >- Accès à la fiche rôle position: 6 data: layout: no_background text: >-

Au clic sur un rôle, l'utilisateur arrive sur la fiche du rôle.
Possibilité de retirer une ou plusieurs personnes de chaque rôle défini.
Un message de confirmation apparait avant le retrait définitif

Possibilité de modifier un rôle : 

  • changer la description du rôle 
  • sélectionner des personnes via une liste déroulante
  • ajouter des personnes
  • retirer des personnes

Supprimer le rôle : un message de confirmation apparait avant la suppression définitive.

Ajouter un rôle : l’utilisateur remplit le champ description. Il peut ensuite ajouter des personnes grâce à une liste déroulante.

  • Informations
    • Nom
    • Téléphone
    • Adresse
    • Code Postal
    • Ville

  • Formations dispensées
    • Selection des formations dispensées en cochant
