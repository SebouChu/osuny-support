Possibilité de rechercher directement par nom et de filtrer par type : Extranet d’une école, d’une formation,  d’un laboratoire ou d’une revue scientifique. Possibilité de réinitialiser la recherche.

L’utilisateur en cliquant arrive sur le site de l’extranet et non pas sur le back office comme sur la partie site Web. Pour accéder au contenu il faut passer par la partie Alumni.

notes: >- alt: >- credit: >- - template: chapter title: >- Modifier position: 4 data: layout: no_background text: >-

L'utilisateur peut modifier les informations essentielles de l'extranet sélectionné avec les champs suivants :

  • Informations
    • Nom
    • Domaine
    • Type d'extranet (école/formation/laboratoire/revue scientifique)
    • Formation (liste déroulante avec toutes les formations rentrées dans l'université)
  • Logo
    • Remplacer un fichier

Possibilité de créer un extranet en remplissant les informations de base : 

  • Informations
    • Nom
    • Domaine
    • Type d'extranet (école/formation/laboratoire/revue scientifique)
    • Formation (liste déroulante avec toutes les formations rentrées dans l'université)
  • Logo
    • Choisir un fichier
