Possibilité de créer un laboratoire en remplissant les champs suivants :

  • Nom
  • Adresse
  • Code postal
  • Ville
  • Pays (liste déroulante)
notes: >- image: id: "ad2755af-55bf-46ca-aff3-db71f7ca075c" file: "ad2755af-55bf-46ca-aff3-db71f7ca075c" alt: >- credit: >- - template: chapter title: >- Accès à la fiche position: 3 data: layout: no_background text: >-

Au clic sur un laboratoire, l’utilisateur.rice a accès à la fiche du laboratoire sélectionné :

  • Accès à la page Axes
  • Possibilité de créer un axe avec les champs : Contenu (Nom, Nom court, Texte) et SEO (Métadescription)
  • Possibilité de modifier les informations liées au laboratoire en cliquant sur "Modifier" en bas à droite
  • Possibilité de supprimer le laboratoire en cliquant sur "Supprimer" en bas à gauche
