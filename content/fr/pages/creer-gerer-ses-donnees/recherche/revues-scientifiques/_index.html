Chaque revue scientifique est composée de volumes, de papiers et de types de papiers. L'utilisateur.rice peut créer autant de volumes, de papiers et de types de papiers qu'il, elle le souhaite.

notes: >- image: id: "2fcc8e4a-6170-4670-a6bf-553b63f9395c" file: "2fcc8e4a-6170-4670-a6bf-553b63f9395c" alt: >- credit: >- - template: chapter title: >- Créer un Volume position: 3 data: layout: no_background text: >-

Un volume contient un ou plusieurs papiers. L'utilisateur.rice peut créer autant de volume souhaité.

Chaque volume est structuré comme ceci : 

  • Un titre
  • Un résumé
  • Un texte
  • Des informations (Numéro, publié ou non, date de publication, mots-clés)
  • Seo : méta-description
  • Image : choix d'une image (ordinateur ou banques d'images libres de droits, unsplash, pexels)
notes: >- image: id: "305bafa2-6bdc-4593-b92a-b69af80ad59d" file: "305bafa2-6bdc-4593-b92a-b69af80ad59d" alt: >- credit: >- - template: chapter title: >- Accès à un volume position: 4 data: layout: no_background text: >- notes: >-

L'utilisateur.rice a accès à la liste des papiers présents dans le volume. Sur chaque papier il peut faire des  modifications ou le supprimer.

Il, elle peut également modifier les informations liées au volume en cliquant sur "Modifier" en bas de la page, et supprimer le volume.

image: id: "4b29231c-4ed7-47be-85ec-c0fdab9bfc70" file: "4b29231c-4ed7-47be-85ec-c0fdab9bfc70" alt: >- credit: >- - template: chapter title: >- Modifier un volume position: 5 data: layout: no_background text: >-

L'utilisateur peut modifier les informations liées au volume choisi. 

  • Le titre
  • le résumé
  • le texte
  • Les informations (Numéro, publié ou non, date de publication, mots-clés)
  • Le Seo : méta-description
  • L'image : choix d'une image (ordinateur ou banques d'images libres de droits, unsplash, pexels)
notes: >- image: id: "861828c5-1591-4eb4-9baa-6b951c595014" file: "861828c5-1591-4eb4-9baa-6b951c595014" alt: >- credit: >- - template: chapter title: >- Créer un papier position: 6 data: layout: no_background text: >-

L'utilisateur.rice peut créer un papier en entrant les informations suivantes : 

  • Un titre
  • Un résumé
  • Un extrait
  • Possibilité d'insérer un pdf
  • Possibilité de rentrer le DOI (Digital Object Identifier)
  • Possibilité de relier le papier à un volume existant
  • Publié ou non
  • Date d'acceptation et date de réception
  • Mots-clés
  • Choix des auteur.e.s (cases à cocher ou possibilité de taper les auteur.e.s directement dans un champ dédié)
  • Choix du type de papier, si ceux-ci ont été créés
  • Seo (méta-description)
