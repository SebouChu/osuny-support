Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

image: id: "e89ca143-5173-482d-bde4-2a982f8cd83e" file: "e89ca143-5173-482d-bde4-2a982f8cd83e" alt: >- credit: >- - kind: heading title: >- Choisir le bloc Formations position: 1 level: 2 - kind: block template: image title: >- position: 2 data: text: >- image: id: "a5815e38-de74-4b70-a63b-7f8644be5d59" file: "a5815e38-de74-4b70-a63b-7f8644be5d59" alt: >- credit: >- - kind: heading title: >- Remplir le bloc position: 2 level: 2 - kind: block template: chapter title: >- position: 3 data: layout: no_background text: >-

L'utilisateur peut ajouter une liste de formation en cliquant sur ajouter une formation et ensuite les sélectionner via la liste déroulante.

Les formations doivent au préalable être toutes créées, voir la rubrique formations.
