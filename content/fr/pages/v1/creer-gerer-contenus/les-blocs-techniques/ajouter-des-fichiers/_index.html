Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

L'utilisateur peut ajouter une liste de fichiers téléchargeables, présentés avec leur poids. L'utilisateur rentre les champs suivants pour ajouter un bloc fichiers :

  • Titre
  • Publié ou non
  • Description
  • Ajouter un ou plusieurs fichiers : titre / choisir un fichier
  • Possibilité de retirer un ou plusieurs fichiers
