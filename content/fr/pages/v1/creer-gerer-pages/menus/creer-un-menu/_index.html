L'utilisateur peut créer un nouvel élément dans le menu en rentrant un titre et choisir un type pour cet élément : 

  • Titre intermédiaire
  • Url
  • Page spécifique

Une fois l'élément créé, il va le retrouver dans l'arborescence de son menu sur la page d'accueil du menu dans "Menu principal".

Pour que dans le menu, un sous menu apparaisse en redirigeant vers une page du site, il faut choisir comme type "Page spécifique". Il faut ensuite choisir la cible (la page de redirection). Un titre par défaut apparait (celui de la page de redirection), l'utilisateur peur changer le nom du titre de l'élément créé.
