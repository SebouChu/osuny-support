Ce bloc vous permet d'intégrer des vidéos depuis les plateformes Viméo, Youtube et Dailymotion.

notes: >- alt: >- credit: >- - kind: heading title: >- 1/ Cliquez sur Ajouter un bloc position: 1 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "44cb636e-27d1-4602-82a4-29112c821638" file: "44cb636e-27d1-4602-82a4-29112c821638" alt: >- credit: >- - kind: heading title: >- 2/ Cliquez sur le bloc Vidéos position: 2 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Cliquez sur le bloc vidéos pour l'ajouter.

notes: >- image: id: "c3a9bb18-122c-4a4b-b974-b1a6020d25e8" file: "c3a9bb18-122c-4a4b-b974-b1a6020d25e8" alt: >- credit: >- - kind: heading title: >- 3/ Remplissez le bloc position: 3 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Remplissez le bloc en ajoutant un nom, l'URL de la vidéo, un titre et une transcription pour être en conformité concernant l'accessibilité des contenus.
