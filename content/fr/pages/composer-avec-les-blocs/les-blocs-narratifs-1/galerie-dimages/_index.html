Ce bloc va vous permettre d'ajouter une galerie d'images, avec pour chaque image une description accessible, un texte et un crédit.

notes: >- alt: >- credit: >- - kind: block template: gallery title: >- position: 2 data: description: >- layout: grid images: - kind: heading title: >- Cliquez sur Ajouter un bloc position: 1 level: 2 - kind: block template: chapter title: >- position: 1 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Cliquez sur le bloc galerie pour l'ajouter.
