Le bloc fichiers vous permet d'ajouter une liste de termes et leur définition.

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Cliquez sur le bloc définitions pour l'ajouter.

notes: >- image: id: "e440db78-dca7-443b-b3f5-17cd802fb525" file: "e440db78-dca7-443b-b3f5-17cd802fb525" alt: >- credit: >- - kind: heading title: >- Remplissez le bloc position: 3 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Rédigez le contenu en ajoutant un nom et une description.

notes: >- image: id: "31a81bdd-5782-4161-b092-0b238332f6ab" file: "31a81bdd-5782-4161-b092-0b238332f6ab" alt: >- credit: >- - kind: heading title: >- Ajouter des définitions position: 4 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Cliquez sur "Ajouter une définition" pour ajouter un titre et un texte. Vous pouvez ajouter autant de définitions que vous le souhaitez.
