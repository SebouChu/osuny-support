Le bloc fichiers vous permet d'ajouter une liste de fichiers téléchargeables, présentés avec leur poids.

notes: >- alt: >- credit: >- - kind: heading title: >- Cliquez sur Ajouter un bloc position: 1 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Cliquez sur le bloc fichiers pour l'ajouter.

notes: >- image: id: "3617e8e2-1537-4782-bf0c-45624b645c5a" file: "3617e8e2-1537-4782-bf0c-45624b645c5a" alt: >- credit: >- - kind: heading title: >- Remplissez le bloc position: 3 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Rédigez le contenu en ajoutant un nom et une description.
