Le bloc de liste organisations va vous permettre de mettre en avant une liste d'organisations, avec leur logo, leur site et leur nom.

notes: >- alt: >- credit: >- - kind: heading title: >- Cliquez sur Ajouter un bloc position: 1 level: 2 - kind: block template: chapter title: >- position: 1 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "ce86228a-f8a3-40b2-b1ee-04955f985a1c" file: "ce86228a-f8a3-40b2-b1ee-04955f985a1c" alt: >- credit: >- - kind: heading title: >- Cliquez sur le bloc organisations position: 2 level: 2 - kind: block template: chapter title: >- position: 1 data: layout: no_background text: >-

Cliquez sur le bloc Organisations pour l'ajouter.

notes: >- image: id: "7e627b7b-c16e-4ca9-a8e9-e6fa6635f718" file: "7e627b7b-c16e-4ca9-a8e9-e6fa6635f718" alt: >- credit: >- - kind: heading title: >- Choisissez la mise en page position: 3 level: 2 - kind: block template: chapter title: >- position: 1 data: layout: no_background text: >-

Vous pouvez choisir la mise en page du bloc : une mise en page en grille ou en carte.

notes: >- image: id: "c674a6ae-dfd6-4c70-9bfc-8f22061e3020" file: "c674a6ae-dfd6-4c70-9bfc-8f22061e3020" alt: >- credit: >- - kind: heading title: >- Choisissez les options d'affichage position: 4 level: 2 - kind: block template: chapter title: >- position: 1 data: layout: no_background text: >-

Vous pouvez choisir les options d'affichage de la liste de pages affichée : avec des liens cliquables vers les pages des organisations et/ou par ordre alphabétique.
