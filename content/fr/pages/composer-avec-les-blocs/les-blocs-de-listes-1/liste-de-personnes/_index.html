Le bloc de liste personne va vous permettre de mettre en avant une liste de personnes, avec photo, fonction et lien vers la page de la personne.

notes: >- alt: >- credit: >- - kind: heading title: >- Cliquez sur Ajouter un bloc position: 1 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "c85bafdf-f020-4464-89cb-63c979561ef8" file: "c85bafdf-f020-4464-89cb-63c979561ef8" alt: >- credit: >- - kind: heading title: >- Cliquez sur le bloc personnes position: 2 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Cliquez sur le bloc Personnes pour l'ajouter.

notes: >- image: id: "80bc9dd2-a5ef-4c9a-857e-621d4b9f4f38" file: "80bc9dd2-a5ef-4c9a-857e-621d4b9f4f38" alt: >- credit: >- - kind: heading title: >- Choisissez les options d'affichage position: 3 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Vous pouvez choisir les options d'affichage de la liste de pages affichée : avec des liens cliquables vers les pages des organisations, avec un portrait de chaque personne et/ou par ordre alphabétique.
