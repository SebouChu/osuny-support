Le bloc chapitre va vous permettre de mettre en avant une sélection de pages, avec plusieurs options d'affichage.

notes: >- alt: >- credit: >- - kind: heading title: >- Cliquez sur Ajouter un bloc position: 1 level: 2 - kind: block template: chapter title: >- position: 1 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "945531bf-e140-4541-aa0c-d7428de5c5de" file: "945531bf-e140-4541-aa0c-d7428de5c5de" alt: >- credit: >- - kind: heading title: >- Cliquez sur le bloc Pages position: 2 level: 2 - kind: block template: chapter title: >- position: 1 data: layout: no_background text: >-

Cliquez sur le bloc pages pour l'ajouter.

Vous pouvez choisir la mise en page du bloc : une mise en page en grille, en liste, en cartes ou en alternance.

notes: >- image: id: "6b5d0d99-beb3-4efb-b57a-ab94f905bd48" file: "6b5d0d99-beb3-4efb-b57a-ab94f905bd48" alt: >- credit: >- - kind: heading title: >- Choisissez les options d'affichage position: 1 level: 2 - kind: block template: chapter title: >- position: 2 data: layout: no_background text: >-

Vous pouvez choisir les options d'affichage de la liste de pages affichée : afficher une courte description de la page principale, afficher des courtes descriptions des pages sélectionnées, ou encore afficher les images des pages.
