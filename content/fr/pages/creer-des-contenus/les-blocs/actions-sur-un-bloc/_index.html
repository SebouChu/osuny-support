Les éléments essentiels pour constituer une page.

elements: - title: >- Bloc Chapitre description: >- Une section de document avec un titre, un texte, des notes de bas de page et une image. - title: >- Bloc Image description: >- Une image avec sa description alternative accessible, son texte et son crédit. - title: >- Bloc Vidéo description: >- Une vidéo intégrée dans la page depuis diverses plateformes, avec la transcription et sans lecture automatique. - title: >- Bloc Tableau description: >- Un tableau de données, aussi responsive et accessible que possible. ---
