L'utilisateur a accès aux éléments qui constituent le menu sélectionné. Il peut ensuite choisir de modifier, supprimer un élément du menu ou de créer un nouvel élément.

notes: >- image: id: "97eb01fc-1c99-4634-9d63-69c09d1030fd" file: "97eb01fc-1c99-4634-9d63-69c09d1030fd" alt: >- credit: >- - template: chapter title: >- Créer un élément du menu position: 3 data: layout: no_background text: >-

L'utilisateur peut créer un nouvel élément dans le menu en rentrant un titre et choisir un type pour cet élément : 

  • Titre intermédiaire
  • Url
  • Page spécifique

Une fois l'élément créé, il va le retrouver dans l'arborescence de son menu sur la page d'accueil du menu dans "Menu principal".

Pour que dans le menu, un sous menu apparaisse en redirigeant vers une page du site, il faut choisir comme type "Page spécifique". Il faut ensuite choisir la cible (la page de redirection). Un titre par défaut apparait (celui de la page de redirection), l'utilisateur peur changer le nom du titre de l'élément créé.

L'utilisateur modifie juste les informations essentielles ici : le titre et le type de menu choisi.

notes: >- image: id: "d8363d51-08ce-48e4-a252-e56284444011" file: "d8363d51-08ce-48e4-a252-e56284444011" alt: >- Modifier le menu credit: >- - template: chapter title: >- Supprimer un menu position: 6 data: layout: no_background text: >-

L'utilisateur peut supprimer le menu choisi.
Un message de confirmation apparait avant la suppression définitive.
