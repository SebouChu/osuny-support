Pour créer une actualité, l'utilisateur doit cliquer sur le bouton "créer" en bas à droite ou "nouvelle curation" s'il s'agit d'un flux RSS.

image: id: "f084bcd7-c4d4-4f2f-9dfc-cfe94b1beb2a" file: "f084bcd7-c4d4-4f2f-9dfc-cfe94b1beb2a" alt: >- Image Actualités credit: >- - template: chapter title: >- Créer une actualité position: 2 data: layout: no_background text: >-

Au clic sur "créer", l'utilisateur arrive sur une page vide pour créer une actualité avec les champs suivants : 

  • Contenu :  Titre (obligatoire) / Chapô
  • Catégorie : Choix d'une ou plusieurs catégories définies auparavant
  • SEO :  Meta Description
  • Informations : Publié ou non / Mis en avant ou non / Date de publication / Auteur.rice / Slug
  • Image :  Choisir un fichier / Texte alternatif / Crédit / Chercher une image (Unsplash ou Pexels)

Il clique sur enregistrer pour valider l'actualité.
Possibilité de choisir de publier la page ou non et donc de la mettre en brouillon.

Une fois l'actualité créée, l'utilisateur peut remplir l'actualité par les blocs de contenus en cliquant sur la fiche de l'actualité, sur son nom (Voir Rubrique "Blocs de contenus")

L'utilisateur peut prévisualiser une actualité sous mobile, tablette ou desktop.

Au clic sur "nouvelle curation", l'utilisateur a la possibilité d'ajouter une url dans le but de récupérer les nouvelles actualités automatiquement via un flux RSS.

Possibilité de modifier le contenu, les informations, la catégorie, le SEO et l’image de l’actualité.

Nous ne pouvons pas ajouter, ni modifier des blocs de contenus en cliquant sur Modifier, pour cela il faut cliquer sur le titre de l’actualité.

  • Contenu :  Titre (obligatoire) / Chapô
  • Catégorie : Choix d’une ou plusieurs catégories
  • SEO :  Meta Description
  • Informations : Publié ou non / Mis en avant ou non / Date de publication / Auteur.rice / Slug
  • Image :  Choisir un fichier / Texte alternatif / Crédit / Chercher une image (unsplash ou Pexels)

Accès a la fiche de l'actualité avec les informations suivantes :

  • Informations essentielles  :
    - Contenu (modifiable seulement en cliquant sur “Modifier” en bas de page) : Chapô / Texte / Médias
    - Catégorie (modifiable seulement en cliquant sur “Modifier” en bas de page). Disposition d’un lien direct vers la ou les catégories avec toutes les actualités liées à la catégorie choisie
    - Informations (modifiable seulement en cliquant sur “Modifier” en bas de page) :  Statut publié ou non avec la date / Statut mis en avant ou non / Auteur.rice (accès direct à tous les articles de l’auteur.rice) / Slug
  • Blocs de contenus : Ils permettent de rajouter des blocs dans une actualité (texte, image, vidéo, chiffre, etc.). Pour ajouter, modifier un bloc de contenu, c’est ici et non pas quand nous cliquons sur “modifier” une actualité. Voir Rubrique “Blocs de contenus”
     - Modifier : L’utilisateur arrive sur le bloc de contenu choisi pour le modifier.
     - Dupliquer : L’utilisateur peut dupliquer le bloc de contenu choisi.
  • Modifier : L’utilisateur peut modifier les infos génériques de l’actualité. Voir sous-rubrique "Modifier une actualité".
  • Prévisualiser : L’utilisateur peut prévisualiser l’actualité avec les blocs de contenus sous format mobile, tablette ou desktop. Voir sous-rubrique "Prévisualisation".
  • Supprimer : L’utilisateur peut supprimer l’actualité. Voir sous-rubrique "Supprimer une actualité".
  • Fichier statique : L’utilisateur peut créer un fichier statique.

L’utilisateur peut filtrer les actualités pour accéder plus facilement à l'actualité souhaitée. Il peut filtrer : 

  • par mot clé
  • par auteur.rice
  • par catégorie
  • par mise en avant

L’utilisateur clique sur “Filtrer” pour faire apparaitre la liste filtrée. L’utilisateur peur réinitiailiser la liste en cliquant sur le bouton “Réinitialiser”

L'utilisateur a la possibilité de supprimer une actualité.
Un message de confirmation apparait avant la suppression définitive.

L'utilisateur peut gérer les catégories créées pour les actualités. Au clic il accède aux informations génériques de la catégorie.

  • Contenu :  Chapô 
  • Informations : slug /catégories enfants (Cet élément est cliquable et nous arrivons sur la liste des actualités de la catégorie enfant sélectionnée)
  • Modifier : voir sous-rubrique "Modifier une catégorie"
  • Supprimer : voir sous-rubrique "Supprimer une catégorie".
  • Fichier statique

Au clic sur un.e auteur.rice, accès à toutes ses actualités. Le classement se fait par ordre alphabétique.

  • Contenu : E-mail / Description courte / Biographie
  • Informations : Slug / Utilisateur Voir Rubrique  "Utilisateur.rice.s"
  • Photo de profil 
  • SEO : Meta description
  • Réseaux Sociaux : Site web / Linkedin (url)
  • Actualités : Liste des actualités de l’Auteur.rice / Accès à l’actualité (Voir sous-rubrique "Clic sur l'actualité" )/ Accès à la catégorie (Voir rubrique "Clic sur la catégorie") / Modifier (Voir sous-rubrique "Modifier une catégorie") /  Supprimer (Voir sous-rubrique "Supprimer une actualité")
  • Modifier : Voir Rubrique Tableau de bord/Université/Personnes / Auteur.rice/Nom de la personne
