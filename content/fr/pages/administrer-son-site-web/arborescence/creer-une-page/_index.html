L'utilisateur a accès à l'arborescence du site avec les dossiers et les sous dossiers.

À côté du nom du dossier est indiqué "ouvrir le dossier". Au clic les sous dossiers et/ou sous pages apparaissent.

Pour ajouter une page l'utilisateur doit cliquer sur le bouton créer en bas à droite dans l'arborescence.

image: id: "37386963-ca08-4b5b-8090-bca86cdfde86" file: "37386963-ca08-4b5b-8090-bca86cdfde86" alt: >- credit: >- - template: chapter title: >- Remplir une page position: 2 data: layout: no_background text: >-

Création d'une page dans l'arborescence en remplissant les champs suivants :

  • Contenu : Titre / Nom affiché dans le fil d’Ariane / Résumé/ Titre affiché dans le header
  • Informations : Publié ou non / Slug / Page parente : liste déroulante / Classe CSS spécifique / Pleine largeur ou non
  • SEO  : Méta description
  • Image : Choisir un fichier / Texte alternatif / Crédit / Chercher une image (Unsplash ou Pexels)

Si l'utilisateur intègre une image via Unsplash ou Pexels l'image est crédité automatiquement. Si l'utilisateur télécharge une image de son ordinateur, il peut la créditer manuellement.
