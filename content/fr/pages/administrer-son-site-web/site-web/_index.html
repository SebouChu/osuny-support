L'utilisateur dispose d'un accès direct vers le site en ligne. L'utilisateur dispose d'un accès vers le back office du site d'université parent.

notes: >- alt: >- credit: >- - template: chapter title: >- Dernières actualités position: 3 data: layout: no_background text: >-

Affichages des dernières actualités par défaut.
Au clic sur l'actualité, l'utilisateur accède aux informations de l'actualité. Il peut modifier, supprimer également directement depuis cette partie une actualité.

L'utilisateur peut également créer une actualité ou une nouvelle curation. Pour plus de détails voir la Rubrique Actualités.

notes: >- alt: >- credit: >- - template: chapter title: >- Dernières pages créées position: 4 data: layout: no_background text: >-

Affichage des dernières pages créées par défaut.

L'utilisateur peut depuis cette partie créer une nouvelle page, modifier ou supprimer une page existante. 
Pour plus de détails voir la Rubrique Arborescence.

notes: >- alt: >- credit: >- - template: chapter title: >- Modifier le Site Web position: 5 data: layout: no_background text: >-

L’utilisateur a accès aux informations de base du site web.

L’utilisateur peut modifier/ajouter les éléments suivants sur le site :

  • Informations : Nom /Url / Type de site / École / Langues 
  • Git : Provider Git / Point d’accès Git / Access Token / Repository
  • Analytics : Plausible Url

L'utilisateur peut importer le site.
L'utilisateur a accès au bouton "Voir l'import du site" une fois l'import fait. Il remplace le bouton "Importer le site".

notes: >- alt: >- credit: >- - template: chapter title: >- Supprimer le Site Web position: 7 data: layout: no_background text: >-

Possibilité de supprimer le site. 
Un message de confirmation apparait avant la suppression définitive.
