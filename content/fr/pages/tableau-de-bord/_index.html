L’utilisateur a accès aux sites web et/ou extranets dont il a la gestion.
L’utilisateur a accès sur le côté aux différentes rubriques pour gérer le contenu via un menu toujours présent.
L’accès au tableau de bord peut se faire depuis n’importe quelle page du back office.

L’accueil pour l’utilisateur est personnalisé.
Est indiqué le nombre de tâches en cours de traitement, non finalisées.

C’est une bonne indication pour l’utilisateur, pour l’inciter à finaliser l’exécution de tâches dans le but de contribuer à diminuer l’empreinte carbone.
