Le tableau de bord est l'écran principal d'administration d'un site web ou d'un extranet. Il résume les informations sur le site ou l'extranet. Il permet d'accéder au back-office et pour les administrateur.rice.s de modifier les paramètres de configuration et de gérer les utilisateur.rice.s.

notes: >- alt: >- credit: >- - template: chapter title: >- Tâches en cours de traitement position: 2 data: layout: no_background text: >-

Le tableau de bord indique les tâches en cours de traitement. La bonne exécution de ces tâches contribue à réduire l'empreinte carbone du numérique en précalculant ce qui peut l'être, ce qui peut générer un délai avant que vos modifications ne soient visibles.

notes: >- alt: >- credit: >- - template: chapter title: >- Le choix du thème position: 3 data: layout: no_background text: >-

Sur le tableau de bord, vous avez le choix entre deux thèmes pour l'administration de votre site ou extranet. Le thème "Pure", très épuré et le thème "Appstack", plus classique. Vous pouvez changer de thème quand vous le souhaitez.
