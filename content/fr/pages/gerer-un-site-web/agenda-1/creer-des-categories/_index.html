Vous avez la possibilité de créer des catégories pour vos événements, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux événements d'une catégorie donnée. Un événement peut être classé dans plusieurs catégories.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer une catégorie position: 1 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Agenda", puis sur "Créer" à côté de l'espace Catégories.

image: id: "9947c715-4959-4557-a9b6-96eaa238965a" file: "9947c715-4959-4557-a9b6-96eaa238965a" alt: >- credit: >- - kind: heading title: >- Remplir les informations de la catégorie créée position: 2 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir une image.

image: id: "856536b7-ef7c-48e8-8192-c1ca7089cc5d" file: "856536b7-ef7c-48e8-8192-c1ca7089cc5d" alt: >- credit: >- - kind: heading title: >- Relier un événement à une catégorie position: 3 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Pour associer un événement à une catégorie, il faut d'abord créer les catégories. Ensuite, allez sur la page de modification des informations de l'événement et sélectionnez la catégorie souhaitée.

image: id: "bb32d0c7-1637-4a9e-9df8-50e9b6135607" file: "bb32d0c7-1637-4a9e-9df8-50e9b6135607" alt: >- credit: >- - kind: heading title: >- Gérer les catégories position: 4 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les événements de la catégorie sélectionnée.
