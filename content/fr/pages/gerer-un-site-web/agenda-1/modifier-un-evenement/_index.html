Vous pouvez modifier les blocs de contenu de votre événement ainsi que les informations de votre événement. Ces actions sont séparées. Pour modifier vos événements, cliquez sur l'onglet "Agenda". Ensuite, vous pouvez accéder à l'événement souhaité en cliquant sur le nom de l'actualité.
