Vous avez la possibilité de créer des catégories pour vos actualités, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux actualités d'une catégorie donnée. Une actualité peut être classée dans plusieurs catégories.

notes: >- alt: >- credit: >- - kind: heading title: >- 1/ Créer une catégorie position: 1 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Actualités, puis sur "Créer"en bas sous l'espace Catégories

notes: >- image: id: "66a57243-dbd2-4115-a529-201858a40e32" file: "66a57243-dbd2-4115-a529-201858a40e32" alt: >- credit: >- - kind: heading title: >- 2/ Remplir les informations de la catégorie créée position: 2 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Il s'agit des informations essentielles à remplir lors de la création d'une nouvelle actualité : le nom de la catégorie, un résumé, et éventuellement une catégorie parente si nécessaire. Vous pouvez également choisir une image pour illustrer votre catégorie.

notes: >- image: id: "3beb3ac7-a3c9-4921-8e09-69c883199618" file: "3beb3ac7-a3c9-4921-8e09-69c883199618" alt: >- credit: >- - kind: heading title: >- 3/ Relier une actualité à une ou plusieurs catégories position: 3 level: 2 - kind: heading title: >- 3/ Gérer les catégories position: 4 level: 2 - kind: block template: chapter title: > position: 2 data: layout: no_background text: >- notes: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à toutes les actualités de la catégorie sélectionnée.
