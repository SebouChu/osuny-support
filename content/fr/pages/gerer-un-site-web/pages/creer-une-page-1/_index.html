Une page vous permet de créer et afficher du contenu sur votre site Web. Vous pouvez créer autant de pages que vous le souhaitez. Après la création d'une page, vous pouvez l'ajouter à votre menu de navigation pour la retrouver facilement.

notes: >- alt: >- credit: >- - template: chapter title: >- 1/ Créer votre page position: 2 data: layout: no_background text: >-

Pour créer une nouvelle page au site web, vous devez cliquez sur l'onglet "Arboresence", puis sur "Créer"en bas à droite.

notes: >- image: id: "7867ce8c-80d4-4c9b-b038-43a0a9779c74" file: "7867ce8c-80d4-4c9b-b038-43a0a9779c74" alt: >- credit: >- - template: chapter title: >- Informations de la page position: 3 data: layout: no_background text: >-

Il s'agit des informations essentielles à remplir lors de la création d'une nouvelle page : le titre de votre page, un résumé, et éventuellement un nom différent pour votre fil d'Ariane si le titre est très long par exemple.

Vous pouvez choisir de publier ou non la page en cochant la case correspondante.

Vous pouvez également décider de lier votre page à une page parente choisie, comme expliqué dans la partie Arborescence.

notes: >- image: id: "2a2b076e-7e01-486d-8f7b-42ae4641536f" file: "2a2b076e-7e01-486d-8f7b-42ae4641536f" alt: >- credit: >- - template: chapter title: >- Pleine largeur position: 4 data: layout: no_background text: >-

Vous avez la possibilité de publier votre page en pleine largeur, ce qui changera l'aspect visuel de votre page.
