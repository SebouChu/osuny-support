Une page vous permet de créer et afficher du contenu sur votre site Web. Vous pouvez créer autant de pages que vous le souhaitez. Après la création d'une page, vous pouvez l'ajouter à votre menu de navigation pour la retrouver facilement.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer une page position: 0 level: 2 - kind: block template: image title: >- position: 2 data: text: >-

Pour créer une nouvelle page au site web, vous devez cliquez sur l'onglet "Pages", puis sur "Créer"en haut.

image: id: "dbb7541e-9953-4562-b978-009fdb9b35f4" file: "dbb7541e-9953-4562-b978-009fdb9b35f4" alt: >- credit: >- - kind: heading title: >- Indiquer les informations de la page créée position: 1 level: 2 - kind: block template: image title: >- position: 3 data: text: >-

Il s'agit des informations essentielles à remplir lors de la création d'une nouvelle page : le titre de votre page, un résumé, et éventuellement un nom différent pour votre fil d'Ariane si le titre est très long par exemple.

Vous pouvez choisir de publier ou non la page en cochant la case correspondante.

Vous pouvez également décider de lier votre page à une page parente choisie, comme expliqué dans la partie Arborescence de ce site support.

image: id: "d3b0be96-7b73-4fe3-a4ac-86a3aa590912" file: "d3b0be96-7b73-4fe3-a4ac-86a3aa590912" alt: >- credit: >- - kind: heading title: >- Choisir le format de la page position: 2 level: 2 - kind: block template: gallery title: >- position: 5 data: description: >-

Vous avez la possibilité de publier votre page en pleine largeur, ce qui changera l'aspect visuel de votre page. Sur de grands écrans, la page en pleine largeur utilisera tout l'espace disponible, ce qui est pertinent pour événementialiser une page.

Si la page n'est pas en pleine largeur, l'espace dédié au contenu sera réduit pour faciliter la lecture, et l'espace libre pourra être utilisé pour une table des matières facilitant la navigation.

Vous pouvez composer votre page à l'aide d'un système de blocs de contenus. Ces blocs vous permettent d'organiser votre page comme vous le souhaitez. Vous pouvez utiliser des blocs de contenu de base tels que des blocs de texte, d'image, de vidéo ou de son, des blocs narratifs avec des galeries de photos, des témoignages ou encore des appels à l'action, des blocs de listes pour faire des liens entre vos pages, ainsi que des techniques pour insérer une page de contact ou encore du code HTML.

Vous pouvez également rajouter des titres et les associer ou non au bloc de contenu souhaité. Cette gestion des titres et des blocs de contenus va vous permettre d'obtenir une composition de vos pages simple et conviviale.

Retrouver l'explication de tous ces blocs de contenus dans la rubrique "Composer avec les blocs".

Une fois les blocs de contenu créés, votre page est organisée comme vous pouvez le voir sur la capture d'écran. Vous pouvez déplacer les différents blocs et titres comme vous le souhaitez et réorganiser votre page.
