La modification des informations de votre page est différente de la modification des blocs de contenu. Ces actions sont séparées : elles ne se font pas sur la même page et l'accès se fait par des boutons différents.

notes: >- alt: >- credit: >- - kind: heading title: >- Modifier les informations d'une page position: 0 level: 2 - kind: block template: image title: > position: 2 data: text: >-

Vous pouvez modifier les informations d'une page depuis l'arborescence en cliquant sur "Modifier".

Vous pouvez modifier les blocs de contenus de la page depuis l'arborescence en cliquant sur le nom de la page et non pas sur le bouton "Modifier".
