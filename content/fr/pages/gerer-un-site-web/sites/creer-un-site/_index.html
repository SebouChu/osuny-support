Une fois votre espace créé, vous avez la possibilité de créer un site web. Plusieurs sites web peuvent être créés dans votre espace.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer un site position: 1 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Pour créer un site web, allez dans la section "Communication"de votre tableau de bord, puis sélectionnez "Site web". Ensuite, cliquez sur le bouton"Créer".
