Une fois votre espace créé, vous avez la possibilité de créer un site web. Plusieurs sites web peuvent être créés dans votre espace.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer un site position: 1 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Pour créer un site web, allez dans la section "Communication"de votre tableau de bord, puis sélectionnez "Site web". Ensuite, cliquez sur le bouton"Créer".

image: id: "3e972995-173d-4696-9b18-3f7b1a5476ec" file: "3e972995-173d-4696-9b18-3f7b1a5476ec" alt: >- credit: >- - kind: heading title: >- Remplir les informations de création d'un site web position: 2 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

ll s'agit des informations à remplir lors de la création d'un site web. Commencez par la partie éditoriale, qui concerne le contenu du site. Indiquez le nom du site et activez les fonctionnalités souhaitées : actualités, agenda, portfolio, etc. Ensuite, choisissez les langues souhaitées pour votre site et définissez la langue par défaut. Pour l'apparence de votre site, vous pouvez sélectionner une image par défaut pour vos vignettes de pages si vous n'en avez pas. De même, choisissez l'image de partage par défaut. L'option agenda vous permet de régler le fuseau horaire. Enfin, dans la section Réseaux sociaux, renseignez les URLs des réseaux sociaux de votre site web.

image: id: "7f20d65c-5f48-4faa-89e2-c0abf975a4ac" file: "7f20d65c-5f48-4faa-89e2-c0abf975a4ac" alt: >- credit: >- - kind: block template: image title: >- ranks: self: false data: text: >- image: id: "4d7f2d20-a585-4413-aeaa-6717c4e5b210" file: "4d7f2d20-a585-4413-aeaa-6717c4e5b210" alt: >- credit: >-

Après la partie éditoriale, remplissez la partie technique qui concerne les réglages techniques.
