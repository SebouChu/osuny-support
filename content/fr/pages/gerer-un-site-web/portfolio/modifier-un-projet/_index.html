Vous pouvez modifier les blocs de contenu de votre projet ainsi que les informations liées. Ces actions sont séparées. Pour modifier vos projets, cliquez sur l'onglet "Portfolio". Ensuite, vous pouvez accéder au projet souhaité en cliquant sur le nom du projet.
