Possibilité de faire directement une recherche et de filtrer les personnes par rôle : Enseignant.e, Personnel administratif, Alumnus, Titulaire, Chercheur.se ou HDR.
Possibilité de réinitialiser le filtre.

notes: >- image: id: "eaffddf4-43ba-4489-ab94-89a84b1a58c1" file: "eaffddf4-43ba-4489-ab94-89a84b1a58c1" alt: >- Filtre rôle credit: >- - template: chapter title: >- Accès à la fiche position: 3 data: layout: no_background text: >-

Au clic sur le nom de la personne, accès à la fiche de cette personne avec ses informations.

Possibilité de supprimer, de modifier la fiche et de générer un fichier statique :

  • Supprimer :  L’utilisateur a la possibilité de supprimer une personne. Un message de confirmation apparait avant la suppression définitive. 
  • Fichier statique
  • Modifier —> Accès à la rubrique “Universités/Personnes/Modifier”
notes: >- image: id: "68beba95-1777-492c-8b27-bb767be1ea56" file: "68beba95-1777-492c-8b27-bb767be1ea56" alt: >- Informations fiche personne credit: >- - template: chapter title: >- Créer position: 4 data: layout: no_background text: >-

L'utilisateur peut créer une fiche pour une personne avec les champs suivants :

  • Informations essentielles : Prénom / Nom de famille / Genre / Anniversaire / Description courte / Biographie
  • Informations : Slug / Utilisateur
  • Coordonnées : Email / Téléphone mobile / Téléphone professionnel / Téléphone personnel / Adresse / Code postal / Ville / Pays
  • Photo de profil : Remplacer le fichier
  • SEO : Méta Description
  • Réseaux Sociaux : Site web / Linkedin(url) / Twitter (nom d'utilisateur)
  • Responsabilités : Enseignant.e / Personnel administratif / Alumnus / Titulaire / Chercheur.se / Peut diriger des recherches (HDR)
