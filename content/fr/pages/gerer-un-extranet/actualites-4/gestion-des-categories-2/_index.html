Vous avez la possibilité de créer des catégories pour vos actualités, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux actualités d'une catégorie donnée.

notes: >- alt: >- credit: >- - kind: heading title: >- 1/ Créer une catégorie position: 1 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Actualités, puis sur "Créer"en bas à côté de l'espace Catégories.

notes: >- image: id: "86aef20f-f715-4592-bb5d-4276deecb1c3" file: "86aef20f-f715-4592-bb5d-4276deecb1c3" alt: >- credit: >- - kind: heading title: >- 2/ Remplir les informations de la catégorie créée position: 2 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir un identifiant pour l'URL de cette catégorie.

notes: >- image: id: "7cfbe481-c4a6-43d2-90d9-bc12924c77b7" file: "7cfbe481-c4a6-43d2-90d9-bc12924c77b7" alt: >- credit: >- - kind: heading title: >- 3/ Relier une actualité à une catégorie position: 3 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Pour associer une actualité à une catégorie, il faut d'abord créer les catégories. Ensuite, allez sur la page de modification des informations de l'actualité et sélectionnez la catégorie souhaitée dans la liste déroulante.

notes: >- image: id: "23e53fce-fb13-424b-ba8a-69b8426a6d45" file: "23e53fce-fb13-424b-ba8a-69b8426a6d45" alt: >- credit: >- - kind: heading title: >- 4/ Gérer les catégories position: 4 level: 2 - kind: block template: chapter title: > position: 3 data: layout: no_background text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à toutes les actualités de la catégorie sélectionnée.
