Vous avez la possibilité de créer des catégories pour vos documents, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux documents d'une catégorie donnée.

notes: >- alt: >- credit: >- - kind: heading title: >- 1/ Créer une catégorie position: 1 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Documents, puis sur "Créer" à côté de l'espace Catégories.

notes: >- image: id: "ec8048a6-4a14-4fd3-89a6-7632157e8679" file: "ec8048a6-4a14-4fd3-89a6-7632157e8679" alt: >- credit: >- - kind: heading title: >- 2/ Remplir les informations de la catégorie créée position: 2 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir un identifiant pour l'URL de cette catégorie.

notes: >- image: id: "bece1986-11a2-4e29-ac81-836c2f22dc7a" file: "bece1986-11a2-4e29-ac81-836c2f22dc7a" alt: >- credit: >- - kind: heading title: >- 3/ Relier un document à une catégorie position: 3 level: 2 - kind: block template: gallery title: > position: 2 data: description: >- layout: grid images: - id: "e421d370-9330-42bf-8e64-fb6a5f90f80d" file: "e421d370-9330-42bf-8e64-fb6a5f90f80d" alt: >- credit: >- text: >- Pour associer une document à une catégorie, il faut d'abord créer les catégories. Ensuite cliquez sur "modifier" sur le document choisi. - id: "c0fd93ee-9af9-43b1-9052-9119c24eb657" file: "c0fd93ee-9af9-43b1-9052-9119c24eb657" alt: >- credit: >- text: >- Sélectionnez la catégorie choisie dans la liste déroulante pour la lier au document. - kind: heading title: >- 4/ Gestion des catégories position: 4 level: 2 - kind: block template: chapter title: > position: 1 data: layout: no_background text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les documents de la catégorie sélectionnée.
